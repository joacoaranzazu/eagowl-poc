# EAGOWL-POC Branding Guidelines

## Logo Usage

The EAGOWL-POC logo should be used consistently across all applications and documentation.

### Color Variants

1. **Primary Logo (Color)**: #00ff88 (EAGOWL Green) on dark background
2. **Monochrome White**: White on dark backgrounds  
3. **Monochrome Black**: Black on light backgrounds

### Logo Formats Required

#### Mobile App
- `icon.png` (1024x1024) - App Store submission
- `adaptive-icon.png` (1024x1024) - Android adaptive icon
- `favicon.png` (16x16, 32x32, 48x48) - Web favicon
- `splash.png` (1242x2436) - iOS splash screen

#### Desktop Console
- `icon.ico` (Windows) - Multiple sizes embedded
- `icon.icns` (macOS) - Multiple sizes embedded  
- `icon.png` (512x512) - Linux desktop icons
- `icon_16x16.png` - Windows taskbar
- `icon_32x32.png` - Windows taskbar
- `icon_48x48.png` - Windows taskbar
- `icon_256x256.png` - macOS dock

#### Web Interface
- `logo-header.png` (200x50) - Navigation header
- `logo-login.png` (300x300) - Login screen
- `logo-dark.png` (400x100) - Dark theme
- `logo-light.png` (400x100) - Light theme

### Usage Guidelines

1. **Minimum Size**: Never use logo smaller than 24x24 pixels
2. **Clear Space**: Maintain minimum clear space equal to logo height
3. **Backgrounds**: Use appropriate variant for background color
4. **Modifications**: Do not modify colors, proportions, or add effects
5. **Consistency**: Use same logo style across all touchpoints

### Color Palette

- **Primary Green**: #00ff88 (Hex), RGB(0, 255, 136)
- **Dark Background**: #1a1a1a (Hex), RGB(26, 26, 26)  
- **Light Background**: #ffffff (Hex), RGB(255, 255, 255)
- **Text Primary**: #ffffff (Hex), RGB(255, 255, 255)
- **Text Secondary**: #b0b0b0 (Hex), RGB(176, 176, 176)

### Typography

- **Primary Font**: Roboto (for UI text)
- **Secondary Font**: Source Code Pro (for code/metadata)
- **Headings**: Roboto Medium 600 weight
- **Body Text**: Roboto Regular 400 weight

### Iconography

- **Style**: Material Design icons filled style
- **Color**: Primary green for actions, white/grayscale for neutral
- **Size**: Use 24dp standard size with appropriate scaling
